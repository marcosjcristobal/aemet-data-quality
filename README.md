# AEMET Data Quality

Proyecto de calidad de datos meteorológicos (Python + SQL).
Objetivo: ingestión (CSV de ejemplo), limpieza, banderas de outliers y límites físicos, KPIs y visualización.

## Estructura
- notebooks/: cuadernos Jupyter
- src/: código Python (utilidades, checks, gráficos)
- data/raw y data/processed: datos originales y limpios (no se suben)
- sql/: consultas de ejemplo
